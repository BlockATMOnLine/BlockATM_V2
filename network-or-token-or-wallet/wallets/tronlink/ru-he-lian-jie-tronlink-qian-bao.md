# 如何连接 TronLink 钱包

### 连接 TronLink 浏览器拓展程序

{% stepper %}
{% step %}
### 准备好 TronLink 钱包

* 下载并安装 TronLink 钱包浏览器拓展程序
* 创建 TronLink 钱包

详情见：[如何创建 TronLink 钱包](ru-he-chuang-jian-tronlink-qian-bao.md)
{% endstep %}

{% step %}
### 在 BlockATM 点击 连接钱包 按钮

* 点击“连接钱包”后，会弹出一个钱包选择窗口。
* 选择网络 Tron
* 选择“TronLink”。
* 如果未自动弹出，选择“TronLink”后，浏览器会显示 TronLink 的连接请求。
{% endstep %}

{% step %}
### 确认连接

* TronLink 会弹出连接请求窗口。
* 点击“连接”。
{% endstep %}

{% step %}
### 连接成功
{% endstep %}
{% endstepper %}
