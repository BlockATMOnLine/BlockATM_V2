# 安全收币

BlockATM 通过企业自托管的[收币合约](shou-bi-zhi-neng-he-yue.md)为企业提供加密货币收款能力，用户通过连接钱包授权智能合约交易或者通过扫码转账的方式交易，交易的加密资产直接转入企业的收币合约，有且只有企业指定的"授权签名地址"可以从合约中提取资产到指定的"资产接收地址"，除此之外任何中心化机构都无权限提取。
